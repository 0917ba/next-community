## Using Tech Stacks
### Basis
- Next.js 13 (App Router)
- TypeScript
### Frontend
- TailwindCSS
### Backend
- Next.js 13
- MongoDB
- Next-Auth (Social Login, Email Login with JWT)

## Features
- Discussion board (using MongoDB CRUD)
- Authentication using Email or Google
- ... and much more
